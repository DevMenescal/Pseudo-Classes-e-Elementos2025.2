Conhecendo Pseudo Elementos e Pseudo Classes.

Pseudo Elementos

Os pseudo elementos são utilizados para estilizar partes específicas de um elemento. Eles criam elementos virtuais que não existem no HTML, permitindo aplicar estilos a partes como, a primeira letra de um texto, ou a primeira linha de um parágrafo. Alguns exemplos incluem:

::before

Permite inserir conteúdo ANTES do conteúdo do elemento selecionado.

Lorem ipsum dolor sit, amet consectetur adipisicing elit. Deleniti neque eos alias, laudantium, a impedit dolorem laborum quod recusandae libero velit hic eius reiciendis nihil dolore ea, porro quam voluptas!

::after

Permite inserir conteúdo APÓS o conteúdo do elemento selecionado.

Lorem ipsum dolor sit, amet consectetur adipisicing elit. Deleniti neque eos alias, laudantium, a impedit dolorem laborum quod recusandae libero velit hic eius reiciendis nihil dolore ea, porro quam voluptas!

::first-letter

Estilizar a PRIMEIRA LETRA de um elemento de bloco.

Lorem ipsum dolor sit, amet consectetur adipisicing elit. Deleniti neque eos alias, laudantium, a impedit dolorem laborum quod recusandae libero velit hic eius reiciendis nihil dolore ea, porro quam voluptas!

::first-line

Estiliza a PRIMEIRA LINHA de um elemento em bloco.

Lorem ipsum dolor sit, amet consectetur adipisicing elit. Deleniti neque eos alias, laudantium, a impedit dolorem laborum quod recusandae libero velit hic eius reiciendis nihil dolore ea, porro quam voluptas!
